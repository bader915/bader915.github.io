# Badr Albdulmajeed Electrical & Electronic Engineering Portfolio

This is a static professional portfolio website designed around Electrical and Electronic Engineering. It uses a clean circuit-grid background, electric-blue and copper theme, project filtering, dark mode, controlled scroll reveal animation, Programme Outcome SWOT graph, E-Portfolio report summary, and contact form mailto support.

## Files

- `index.html` - main portfolio page
- `contact.html` - dedicated contact page
- `styles.css` - complete responsive design and EEE theme
- `script.js` - dark mode, mobile menu, scroll progress, counters, reveal animation, project filters, and contact form behavior
- `assets/Badr_Albdulmajeed_CV.pdf` - downloadable CV PDF
- `assets/Badr_Albdulmajeed_CV.docx` - downloadable CV DOCX

## Notes

The site is static and can be published directly with GitHub Pages. It does not require Tailwind, Font Awesome, or any external CDN, so it will still look correct when hosted as a simple static website.

## Customisation

To add a profile image later, place it in `assets/` and add it to the hero or about section. The current design uses an EEE chip/system visual instead of a personal photo to keep the theme more engineering focused.
